  Feature Engineering (в data_service.py)

  ┌──────────────┬─────────────────────────────────────────────────────────────────────────────────┐
  │    Feature    │                                     Формула                                     │
  ├───────────────┼─────────────────────────────────────────────────────────────────────────────────┤
  │ Risk_Score    │ 0.3×academic_pressure + 0.3×financial_stress + 0.2×work_hours − 0.2×sleep_hours │
  ├───────────────┼─────────────────────────────────────────────────────────────────────────────────┤
  │ Pressure_Sum  │ academic + work pressure                                                        │
  ├───────────────┼─────────────────────────────────────────────────────────────────────────────────┤
  │ Sleep_deficit │ max(0, 7 − sleep_hours)                                                         │
  ├───────────────┼─────────────────────────────────────────────────────────────────────────────────┤
  │ Sleep_hours   │ text → float (4.0 / 5.5 / 7.5 / 9.0)                                            │
  └───────────────┴─────────────────────────────────────────────────────────────────────────────────┘

  Як запустити
  
  cd depression_dashboard
  pip install -r requirements.txt

  # Terminal 1 — Flask (автоматично тренує моделі при першому запуску ~30s)
  python run_backend.py
  # → http://localhost:5050
  
  # Terminal 2 — Streamlit
  streamlit run ui/app.py
  # → http://localhost:8501

  Docker

  docker compose up --build

  ML Pipeline: RandomForest (AUC ~0.88) для prediction, KMeans (k=4) для кластеризації, IsolationForest (5% contamination) для anomaly detection — всі моделі зберігаються в models/classifier.pkl
   і автоматично перетренуються якщо файл відсутній.
